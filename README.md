## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites


* npm
  ```sh
  npm install npm@latest -g
  ```
* Angular
  ```sh
  npm install -g @angular/cli
  ```

### Installation


1. Clone the repo
   ```sh
   git clone https://github.com/Fleetfoxe/angular-xml-read-write
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
